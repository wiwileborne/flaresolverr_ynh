## Overview

FlareSolverr is a proxy server to bypass Cloudflare and DDoS-GUARD protection. It starts a proxy server, and it waits for user requests in an idle state using few resources. When some request arrives, it uses a headless browser (Chromium) to open the URL with user parameters and waits until the Cloudflare challenge is solved (or timeout). The HTML code and the cookies are sent back to the user, and those cookies can be used to bypass Cloudflare using other HTTP clients.

### Features

- Bypass Cloudflare challenges automatically
- Supports proxy configuration
- REST API interface
- Low resource usage when idle
- Headless Chromium browser included
